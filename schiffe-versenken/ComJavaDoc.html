Package: controller

Klasse: Com

Die Klasse Com implementiert die Netzwerkkommunikation eines Schiffe-versenken-Spiels. Sie unterstützt den Betrieb als Server oder Client und verarbeitet den Nachrichtenaustausch über TCP-Sockets.

Hauptaufgaben

  • Aufbau von Server- und Client-Verbindungen
  • Senden und Empfangen von Spielnachrichten
  • Verwaltung des Kommunikationszustands
  • Benachrichtigung der Anwendung über Listener-Callbacks

Enum State

Definiert die Kommunikations- und Spielzustände.

Wichtige Methoden

connectAsClient(String host, int port)Verbindet sich mit einem Server.

Interface Listener

Das Interface stellt Ereignis-Callbacks bereit, die von der Anwendung implementiert werden können. Beispiele: onConnected(), onShot(), onAnswer(), onSave(), onDisconnected().

Architektur

Die Klasse verwendet einen separaten Reader-Thread zum kontinuierlichen Empfang von Nachrichten. Eingehende Befehle werden durch handleLine() analysiert und an den Listener weitergeleitet.
