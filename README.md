# Schiffe Versenken (Programmierpraktikum SoSe 2026)

Dieses Projekt ist im Rahmen des Programmierpraktikums im Sommersemester 2026 entstanden. Es handelt sich um eine Java-basierte Umsetzung des Spieleklassikers "Schiffe Versenken".

## Kern-Features
* **Grafische Oberfläche:** Realisiert mit **Java Swing** für die Darstellung des Spielfelds und der Interaktionen.
* **Multiplayer:** Netzwerkfähigkeit durch **Socket-Programmierung** (TCP), um Partien gegen andere Spieler über das Netzwerk zu führen.
* **Spielmodi:** Enthält die Logik für die Schiffsplatzierung, Schussabfolgen und die Siegesermittlung.

## Technische Details
* **Sprache:** Java
* **UI-Toolkit:** Swing
* **Kommunikation:** Java Sockets
* **Build-Struktur:** Standard Java-Projektstruktur
